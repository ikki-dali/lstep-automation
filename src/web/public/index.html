🍪 Cookie設定

Cookie取得手順

  1. 新しいタブで Lステップ を開く
  2. 通常通りログイン（reCAPTCHA対応）
  3. ログイン後、F12キーで開発者ツールを開く
  4. 「Application」タブ → 「Cookies」→「https://manager.linestep.net」
  5. 表の上で右クリック →「Copy all as JSON」
  6. 下の入力欄に貼り付け

※ Cookieは安全に保存され、Lステップへのアクセスにのみ使用されます
